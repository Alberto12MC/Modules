### Testbenchs dependencies

- TerosHDL: <https://www.terostech.com/>

- VUnit

  ```
  sudo pip install vunit_hdl
  ```

- GHDL gcc compiled.

- gtkwave: visualization.

- cocotb

  ```
  sudo pip install cocotb cocotb-test cocotb-coverage
  ```
